# Privacy Policy for ReplyAI

ReplyAI provides AI-generated smart reply suggestions for supported messaging websites such as WhatsApp Web, Telegram Web, and Instagram Web.

## Data Usage

- Message text may be temporarily processed to generate AI reply suggestions.
- User preferences may be stored locally in browser storage.
- ReplyAI does not sell personal data.
- ReplyAI does not collect passwords, payment information, or sensitive personal information.

## Permissions

ReplyAI uses:
- storage permission to save user preferences locally.
- Host permissions to run on supported messaging websites.

## Third-Party Services

ReplyAI communicates with a secure backend server to generate AI responses.
